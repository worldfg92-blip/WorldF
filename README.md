# WorldFG

WorldFG is a modern social networking platform that helps people connect, chat, share stories, upload photos and videos, discover new friends, and promote businesses.

## Features

- User Registration & Login
- Personal Profiles
- Profile Pictures & Cover Photos
- News Feed
- Stories / Status Updates
- Photo & Video Uploads
- Friend Requests
- Follow & Unfollow Users
- Messenger Chat System
- Notifications
- Marketplace
- Business Promotion
- Verification Request System
- Trending Content
- Mobile Friendly Design

## Technology

- HTML5
- CSS3
- JavaScript
- Local Storage Database

## Future Updates

- Real-Time Messaging
- Live Streaming
- Voice Calls
- Video Calls
- Blue Verification Badge
- Business Ads Manager
- Creator Monetization
- Firebase Database Integration

## Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/WorldFG.git
```

2. Open `index.html`

3. Start using WorldFG
